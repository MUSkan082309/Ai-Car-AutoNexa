# AutoNexa 🚗

AI-powered car marketplace built with Next.js, Prisma, Supabase, Clerk Authentication, Gemini AI, and Tailwind CSS.

## Features

* User Authentication with Clerk
* AI-powered car recommendations
* Car listings and detailed car pages
* Test Drive Booking System
* Dealership Management
* Responsive UI
* Supabase PostgreSQL Database
* Prisma ORM Integration

## Tech Stack

* Next.js 15
* React
* Tailwind CSS
* Prisma
* Supabase
* Clerk
* Gemini AI
* Shadcn UI

## Installation

```bash
git clone https://github.com/MUSkan082309/Ai-Car-AutoNexa.git
cd Ai-Car-AutoNexa
npm install
```

Create a `.env` file and add your environment variables.

Run development server:

```bash
npm run dev
```
